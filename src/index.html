Work Experience

Saltside Technologies Frontend developer (2014 - )

Saltside Technologies provides online marketplaces for emerging markets such as Ghana, Bangladesh, Sri Lanka, and Nigeria. All kinds of used goods are sold on these sites - anything from cars, to electronics, to houses!

My responsibility as a frontend developer is to make the marketplace available for as many users as possible, as fast as possible, with the best experience available for the user's device. All under strict network conditions.

To achieve this goal, we use an approach that combines adaptive and responsive design. This enables delivery of a payload optimized for, and tailored to, the user's device.

We also make heavy use of our neat living style guide in our projects. Using a living style guide makes refactoring of styling, and dependant element structure, a breeze.

Tech used: Grunt, Node.js, Express.js, Backbone.js, Less, Webpack, Jade, Selenium, Nightwatch, Vagrant, Cron
